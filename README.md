## Description

Degree37 Backend APIs

* Node v18.16.0  
* Database: PostgreSQL

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Seeder

```bash

# Run seeder
$ yarn run seed

```

## Test

Set ** TEST_DB_NAME ** in .env before running test cases.

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Keycloak Version 
keycloak-21.1.1

## Naming Conventions
** Database **  
Fields should be in ```snake_case```  
Table names in ```snake_case```  

** Code **  
All variables in camelCase
